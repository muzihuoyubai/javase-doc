# 编程基础

### 计算机导论

知己知彼，当我们学习编程语言和计算机交流之前，首先要对计算机有所了解，先在这里认识一下吧。

### C语言

你现在所使用的电脑、手机、平板，无论什么系统，他们的系统内核基本都是C语言开发的。先学C语言再学其他语言这种顺序并不是绝对的。许许多多过来人的经验，学习C语言之后再学习其他语言会更容易、更能理解语言的特性。

### 数据结构和算法

数据结构和算法的威力要在程序开发比较深入的时候才能体现到。这里通过学习数据结构与算法来锻炼我们的逻辑思维能力，方便未来在进行更深层次的技能学习时，能快速掌握好其核心，拥有抽象的能力，比旁人更能容易学会IT技能。

### 数据格式

计算机之间传递数据时，为了方便数据组装和解析需要以某种数据格式保存这些数据。数据格式不能使用word这种专有数据格式，因为必须保证这些数据格式在所有计算机上都可用。这里介绍一些常用的数据格式。

# Web必备

### http协议

《圣经》中有一个通天塔的故事，大致是说，上帝为了阻止人类联合起来，就让人类说不同的语言。人类没法儿沟通，达不成“协议”，通天塔的计划就失败了。如今的世界，被互联网连接在一起，http协议就是贡献力量最多的一种"语言"。

### Nginx

http协议用于连接客户端和服务端，nginx就是服务端的一个程序，客户端对数据的请求都需要经过nginx的调配。可以将网页放到Nginx上，由Nginx发送给客户端的浏览器上展示。

### Html

Html是一种告诉浏览器网页如何显示的语言。你可以利用它来定义网站中的文章内容、标题和数据表，或在页面中嵌入图像和视频，让浏览器知道整个网页的骨架。

### Css

Css是一种能为网页增添样式的语言。它能够修改网页中的字体样式、颜色、背景，增加华丽的动画与3D 效果，是给网页增添设计感的工具。

### JavaScript

JavaScript 最开始是为浏览器设计的一门编程语言，用它编写的程序可以跟随网页加载到浏览器中运行，如今已经成长为网络时代不可或缺的前端顶梁柱，现在已经几乎没有不使用js的网站。同时JavaScript在浏览器外的领域也有很优秀的表现。

## JavaScript技术

### ES6

JavaScript所遵循的语言标准ECMAScript的演化不会停止，每年都会推出新版本。我们通过学习里程碑式的版本ES6开始逐渐了解JavaScript提供的新特性，摸索出学习套路，这样我们就不会害怕版本更新了。

### JQuery

JQuery的核心帮助开发人员简化了DOM操作、Ajax以及浏览器兼容。另外围绕着JQuery还有非常多的插件可供选择。如今的操作DOM的思路发生了变化，但是JQuery思考和处理问题的方式还是值得我们借鉴。

### bootstrap

我想要按钮有个统一样式，到Bootstrap的UI组件中找一找。我想要手机、平板、PC都能展示我的页面并且布局能够适应设备的屏幕，用一用Bootstrap的栅格。另外Bootstrap提供的模块化、API以及大量的第三方插件可以帮助我们快速开发页面。

### nodejs

Node.js是一个JavaScript的编译环境，js代码在可以交给Node.js进行编译和解释，现在js代码终于可以脱离浏览器运行了，依靠Node.js使用JavaScript语言能够开发出功能更强大的应用。

### reactjs

React是一个JavaScript库，专门用于帮助开发人员构建屏幕菜单，搜索栏，按钮以及其他任何与用户交互或展示数据的界面。React使用了全新的方式操作Dom，极大的简化了开发人员的操作。

### vue

Vue是一个JavaScript库，Vue和React很多地方都很类似。Vue 从一开始的定位是尽可能的降低前端开发的门槛。如果想要快速构建一个轻量级(体积小)页面应用，Vue也是一个不错的选择。

## python技术

### python

Python像瑞士军刀一样是个多面手的编程语言。它关系着方方面面，小则可以帮助你解决生活工作中的重复性工作，大则在数据科学和机器学习方面表现得很好，而且在web开发方面也无处不在。学习Python意味着你可以做很多事情，并且为以后学习使用背后强大的类库和工具做准备。

### Django

Django是用 Python 写的一套组件，能帮助你更快、更容易地开发web站点。构建一个web站点时，总需要一些相似的组件：用户注册、登录组件，管理站点的面板，上传文件等等，这些组件都已经被写好，这样无需重新发明轮子就能建立新的站点。

### Scrapy

Python的一个主要的应用是编写爬虫程序。就是通过程序去自动的批量获取web页面上自己想要的数据。Scrapy是Python开发的用于编写爬虫程序的框架，依靠它可以快速编写出爬虫程序，应用于数据挖掘、检测和自动化测试等等领域。

### pandas

 pandas是一种工具，该工具是为了解决数据分析任务而创建的。 pandas提供了大量能使我们快速便捷地处理数据的函数和方法。 你很快就会发现，它是使Python成为强大而高效的数据分析环境的重要因素之一。

### matplotlib

如果你手中有很多数据, 可是不知道该怎么呈现这些数据， Matplotlib能帮你画出美丽的：线图、条形图、柱状图以及3D甚至是图形动画等等

## 数据库

### mysql

MySQL在过去由于性能高、成本低、可靠性好，已经成为最流行的开源关系型数据库管理系统，因此被广泛地应用在中小型网站中。随着MySQL的不断成熟，它也逐渐用于更多大规模网站和应用。

### Oracle

Oracle是甲骨文公司推出的一款关系数据库管理系统。Oracle数据库系统是目前世界上流行的关系数据库管理系统之一，如果数据库用于商业用途，就需要购买相应Oracle产品的License。

### redis

Redis是完全开源免费的，高性能的键值数据库。是当前最热门的NoSql数据库之一，也被人们称为数据结构服务器。Redis将数据存储在内存中，所以读写数据的效率极高。

### mogoDB

MongoDB是一个面向文档的数据库(有别于Mysql关系型数据库和redis键值数据库)，它使用动态模式将数据存储在类似JSON的文档中，其数据结构像是面向对象中的类。

## 区块链

### 区块链基础

从区块链的起源，发展到现状，介绍区块链的基础知识。

### 以太坊DAPP开发

去中心化应用（DAPP），就是一个不存在中心服务器的应用。在网络中成百上千的电脑上，都可以运行该应用的副本，这使得它几乎不可能出现宕机的情况。以太坊则是DAPP运行依赖的环境。

### Rust

Rust语言是一个不需要垃圾回收但是同样可以保证操作内存安全的编程语言。Go、C++、Python、Java、Ruby等等开发语言都被应用在区块链开发，越来越多的著名区块链项目已经选择使用Rust作为其开发语言，Rust语言是一个趋势。

### 以太坊Parity源码

通俗地讲，Parity 用 Rust 语言开发的以太坊节点应用，其特点就是速度块、轻量化。从开发者的角度，Parity是学习以太坊最好的客户端。代码是程序员最好的老师，你需要能读懂源码。本课程以Partiy源码为主线，贯穿以太坊整体架构，让你对整个区块链技术有全局宏观的理解。

### Libra与Move

比特币的目标是"可编程货币"，对"货币"编程。以太坊的目标是"可编程的去中心化应用"，对"应用"编程。 Libra 的设计目标恰好介于二者中间是"可编程资产"。实现一个可以保证资产安全性，又能够提供足够灵活性的系统。Move就是对“资产”编程的语言。

## 大数据

### flume

服务器运行过程中，通常需要采集运行期间输出的日志，汇总后进行分析来获取服务器的运行状态，比如日志中是否有报错记录等等。Flume程序相当于日志文件的快递公司。把数据从数据源收集过来，再将收集到的数据送到指定的目的地。

### kafka

kafka官方定义是分布式的流平台。通俗的理解是连接消息发布者和消息接受者的平台。kafka程序像是一个情报的交易所，发布者将情报放到kafka中，由接受者取走情报。

### hadoop

试想下要如何保存一个1024T大小的文件，又如何分析这个文件内容。由java语言编写的Hadoop框架是专为离线和大规模数据存储、分析而设计的。

 <!--Hadoop的框架最核心的设计就是：HDFS和MapReduce。HDFS为海量的数据提供了存储，则MapReduce为海量的数据提供了计算。-->

### hive

Hive是一个分析海量数据的工具。在Hive看来，hadoop中保存的文件数据都是一张张数据库表。可以使用SQL语句进行检索。

### spark

hadoop本身支持对大数据的分析，但是Spark可以做的更快更好。Spark是一套框架，体系中引入一系列技术可以让处理数据的运算比hadoop快上10倍~100倍

### hbase

HBase 是hadoop database 的缩写，是应用于hadoop之上的数据库工具。主要是为了弥补hadoop在实时操作上的缺失，方便用户可随时操作大规模的数据集。

### Zookeeper

通常hadoop、hbase都是集群搭建，就是同时又多台计算机安装hadoop、hbase环境作为服务器，为了协调这些计算机共通完成工作又不出差错，需要解决服务器间的通讯和协调问题，而zookeeper正是为此而生的。
